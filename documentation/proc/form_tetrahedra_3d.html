public subroutine form_tetrahedra_3d(nk, mesh, tetra, tetracount, tetramap, blocks, indexlist)

Form all the tetrahedra of a 3d FBZ mesh.

nk Number of points in the list of FBZ wave vectors mesh Wave vector grid tetra List of the tetrahedra vertices tetracount Number of tetrahedra in which a wave vector belongs tetramap Wave vector to (tetrahedron, vertex) mapping blocks Is the FBZ wave vector list full or energy restricted? indexlist List of muxed indices of the FBZ wave vectors
