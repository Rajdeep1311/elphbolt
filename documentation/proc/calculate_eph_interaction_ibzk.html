public subroutine calculate_eph_interaction_ibzk(wann, crys, el, ph, num, key)

Parallel driver of g2(k,q) over IBZ electron states.

This subroutine will calculate the full Bloch rep. matrix elements for all the energy window restricted electron-phonon processes for a given irreducible initial electron state = (band, wave vector). This list will be written to disk in files tagged with the muxed state index.

key = 'g', 'X' for vertex, transition probability calculation, respectively.
